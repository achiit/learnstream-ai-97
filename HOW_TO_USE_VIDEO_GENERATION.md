## ğŸ¯ Quick Start

There are **two ways** to generate AI-powered learning videos:

### Method 1: From Dashboard (Recommended)

1. **Go to Dashboard** (`/dashboard`)
2. **Enter your learning topic** in the "What would you like to learn?" box
   - Example: "explain pythagoras theorem"
3. **Click "Generate Learning Content"**
4. You'll be automatically redirected to the Generate page
5. **Click "Start Assessment & Generate Video"**
6. **Answer 3 quick questions** about your knowledge level
7. **Video generation starts automatically!**

### Method 2: Direct from Generate Page

1. **Go to Generate page** (`/generate`)
2. **Enter your prompt** in the textarea
3. **Click "Start Assessment & Generate Video"**
4. **Answer the 3 assessment questions**
5. **Watch your video generate in the background!**

## âš™ï¸ Setup Required (First Time Only)

### 1. Environment Variables

Create `.env` file with:
```env
VITE_GEMINI_API_KEY=your_gemini_api_key
VITE_MANIM_API_URL=http://localhost:5002
```

### 2. Gemini API Key

1. Go to: https://makersuite.google.com/app/apikey
2. Click "Create API Key"
3. Copy and paste into `.env`

### 3. Manim Server

Make sure your Manim video generation server is running on `http://localhost:5002`

Test it:
```bash
curl http://localhost:5002/health
# Should return: {"status": "healthy"}
```

## ğŸ” Troubleshooting

### "Gemini API key not configured"
- Add `VITE_GEMINI_API_KEY` to `.env`
- Restart dev server: `npm run dev`

### "Failed to start video generation"
- Check Manim server is running: `curl http://localhost:5002/health`
- Verify `VITE_MANIM_API_URL` in `.env`

### Questions don't appear
- Check browser console for errors
- Verify Gemini API key is valid
- Check API quota at Google AI Studio

### Video stuck in "started" status
- Normal! Generation takes 1-3 minutes
- Status updates every 5 seconds automatically
- Check Manim server logs for progress

## ğŸ“ Tips

### For Best Results
1. Be specific in your prompts
   - Good: "explain calculus derivatives with examples"
   - Bad: "math stuff"

2. Answer assessment questions honestly
   - AI adapts content to YOUR level
   - Don't guess - accurate assessment = better video

3. Use the queue
   - Generate multiple videos at once
   - No need to wait for one to finish
   - Come back later to view completed ones

### Advanced Usage
- **Review Cycles**: More cycles = higher quality (but slower)
  - Beginners get 4-5 cycles automatically
  - Experts get 1-2 cycles
  
- **Video Queue Persistence**: 
  - Jobs saved in localStorage
  - Can close browser and return later
  - Queue shows on all pages
